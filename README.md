## Getting Started

First, initialize the database with 

```bash
npm run init-db
# or
yarn init-db
# or
pnpm init-db
# or
bun init-db
```

Second run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
